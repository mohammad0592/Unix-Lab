# Short Presentation / Demo Outline

## Slide 1: Project Title

Smart Linux Admin Dashboard

## Slide 2: Project Idea

A Bash terminal dashboard for Linux administration tasks.

## Slide 3: Main Features

- System information
- Users and groups
- File manager
- Permission audit
- Text/log analyzer
- Backup manager
- Full report generator

## Slide 4: Linux Skills Used

- Files and directories
- Permissions
- Users and groups
- Pipes and redirection
- Text processing
- Shell scripting

## Slide 5: Demo

1. Run `./dashboard.sh`
2. Show system information
3. Create a file in the workspace
4. Run permission audit
5. Analyze a sample log file
6. Create a backup
7. Generate a report
